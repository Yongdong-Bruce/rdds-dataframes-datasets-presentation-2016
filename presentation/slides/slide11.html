But we've lost type safety

Using the previous DataFrame (with project and numRequests), what happens if we call collect()? 

      
df.collect() // => yields Array[org.apache.spark.sql.Row]

Unfortunately, Row isn't typesafe: 

      
df.collect().map { row =>
  val project = row(0).asInstanceOf[String]   // Yuck.
  val numRequests = row(1).asInstanceOf[Long] // Yuck.
}
