This code does the same thing the previous RDD code does. Look how much easier it is to read. 

    
val df = parsedRDD.toDF("project", "numRequests")

df.groupBy($"project").
   agg(sum($"numRequests").as($"count")).
   limit(100).
   orderBy($"count").desc.
   show(100)
